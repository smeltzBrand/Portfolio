TeamTracker

TeamTracker

TeamTracker is an Enterprise-level solution, built from the ground up, to streamline project completion through an enhanced ticket service. It can also be used as a defect tracking system. TeamTracker is built with ASP.NET, MVC, C# and SQL. A role based security system was implemented to create permission based roles. The software allows for users to submit tickets and those tickets are tracked through status from New to Resolved.

The system was built with C# and .Net in 3 weeks. The front end was developed with HTML, Javascript and Bootstrap. The back-end utlizies MVC and C#. The database was built with PostgreSQL and hosted on Heroku. Please visit the site and try it out.
